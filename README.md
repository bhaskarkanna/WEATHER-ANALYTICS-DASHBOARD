# Weather Analytics Dashboard

## Quick start

1. Unzip the project and `cd weather-analytics-dashboard`
2. Put your `.env` (already included) or update `VITE_*` vars.
3. Install dependencies:
   ```
   npm install
   ```
4. Run dev server:
   ```
   npm run dev
   ```
5. Open http://localhost:5173

## Notes
- Tailwind/PostCSS config included to avoid `@tailwind` errors.
- This project uses WeatherAPI and optional Firebase (Google sign-in).
- If you get tailwind/postcss errors on Windows, run `npm install` then restart `npm run dev`.
